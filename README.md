# Dizvyn — Premium Dijital Ajans

Dizvyn, modern web teknolojileri, UI/UX tasarım ve butik dijital çözümler sunan kreatif bir dijital ajanstır.

## Özellikler
- 🚀 **Performans Odaklı:** Vite + React ile en üst düzey hız.
- 🎨 **Modern Tasarım:** Framer Motion ile akıcı animasyonlar ve glassmorphism estetiği.
- 📱 **Responsive:** Tüm cihazlarla tam uyumlu arayüz.
- 🔍 **SEO Dostu:** Optimize edilmiş meta etiketleri ve şema yapıları.

## Kurulum

```bash
# Bağımlılıkları yükleyin
npm install

# Geliştirme sunucusunu başlatın
npm run dev

# Üretim için derleme yapın
npm run build
```
